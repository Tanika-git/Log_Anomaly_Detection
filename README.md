# 🔍 Log Anomaly Detection System

## 📌 Overview
This project implements an AI-based log anomaly detection system to identify unusual patterns in system logs using machine learning and deep learning techniques.

---

## ❗ Problem Statement
System logs are large and complex, making it difficult to manually detect anomalies. This project automates anomaly detection using advanced models.

---

## 🎯 Objectives
- Detect anomalies in log data  
- Compare multiple models  
- Improve detection using hybrid approach  

---

## 🧠 Models Used
- Isolation Forest  
- CNN + LSTM  
- Hybrid Model  

---

## ⚙️ Tech Stack
- Python  
- NumPy, Pandas  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib  

---

## 🔄 Workflow
1. Log preprocessing  
2. Encoding & sequence generation  
3. Model training  
4. Anomaly detection  
5. Visualization  

---

## 📊 Results
- Hybrid model improves anomaly detection  
- Better performance than individual models  

---

## 🤖 Why Hybrid Model?
Isolation Forest detects statistical anomalies, while CNN+LSTM captures sequential patterns. Combining both improves accuracy.

---

## 📊 Dataset
HDFS Log Dataset  
https://github.com/logpai/loghub/tree/master/HDFS  

---

## 🚀 How to Run

pip install -r requirements.txt  
python main.py
